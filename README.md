# Staines Super Kings Website

A modern React website for the Staines Super Kings cricket team.

## Features

- 🏏 Professional homepage with team logo
- ℹ️ Team information section (Founded, Mission, Values, Vision)
- 📧 Contact information section (Email, Phone, Location)
- 📱 Responsive design for all devices
- ⚡ Built with React and Vite for fast performance

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ram29112018/StainesSuperKingswebsite.git
cd StainesSuperKingswebsite
```

2. Install dependencies:
```bash
npm install
```

### Running the Application

#### Development Mode
Start the development server:
```bash
npm run dev
```
The website will be available at `http://localhost:5173`

#### Production Build
Build the application for production:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

### Code Quality

Lint the code:
```bash
npm run lint
```
